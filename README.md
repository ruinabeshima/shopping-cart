# Shopping Cart 
- Deployed: https://shopping-cart-eight-iota-76.vercel.app/
- My capstone React project. An application where users can shop for items and add them to their shopping cart. 

## Technologies
- A React + Vue project 
- React Router for client-side routing 
- Component-based CSS Styling
- FakeStore API for shopping items
- Material UI for the modal on Shop Page 

## Features 
- Navigation between Home, Shop and Cart Pages. 
- Button in navigation bar showing number of items in cart. 
- Users can delete items from the cart, and the number of items in cart will be updated accordingly.
